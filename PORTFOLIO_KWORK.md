Адаптивный лендинг «Курсы 3D-моделирования» — HTML/CSS вёрстка

## Краткое описание (для превью)

Сверстал адаптивный одностраничный лендинг образовательной платформы: шапка, баннер, блок преимуществ, карточки преподавателей и курсов, подвал. Корректное отображение на экранах от 320 до 1920+ px.

---

## Полное описание (для карточки портфолио)

### Задача

Сверстать адаптивный лендинг по макету Figma для онлайн-школы 3D-моделирования. Страница должна корректно отображаться на мобильных, планшетах и десктопах.

### Что сделано

- Семантическая HTML5-разметка: `header`, `main`, `section`, `nav`, `footer`
- Адаптивная сетка на Bootstrap Grid (12 колонок)
- 4 брейкпоинта: 320px, 576px, 768px, 1200px
- Адаптивные изображения преподавателей через `<picture>` + `srcset`
- Фоновые изображения курсов с подменой под разные разрешения
- Hover- и focus-состояния для кнопок и ссылок
- Подключены локальные шрифты Roboto (woff2)
- SVG-иконки соцсетей в подвале
- Доступность: `aria-label`, `alt`, видимые состояния фокуса

### Стек

HTML5 · CSS3 · Bootstrap Grid · Flexbox · CSS Variables · BEM-подобная методология

### Результат

Готовая к публикации вёрстка без JavaScript. Лёгкая, быстрая, кроссбраузерная. Подходит как основа для интеграции с CMS или фреймворком.

## Ответы на частые вопросы заказчиков

**Используете ли вы Bootstrap?**
Использую Bootstrap Grid для сетки. Полный фреймворк не подключаю — стили пишу сам, без лишнего веса.

**Делаете ли pixel perfect?**
Да, сверяю с макетом по отступам, шрифтам и цветам. Допустимое отклонение — 1–2 px.

**Нужен ли JavaScript?**
Для статичной вёрстки — нет. При необходимости подключу слайдеры, меню, формы.

**Сдаёте ли исходники?**
Да: HTML, CSS, шрифты, изображения — всё в архиве или репозитории.
