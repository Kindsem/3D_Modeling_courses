# Лендинг «Курсы 3D-моделирования»

Адаптивная одностраничная вёрстка лендинга образовательной платформы. Проект демонстрирует владение **HTML5**, **CSS3**, **Bootstrap Grid** и **адаптивной вёрсткой** под все основные разрешения.

## Демо (GitHub Pages)

`https://kindsem.github.io/3D_Modeling_courses/`

## Технологии

- HTML5 — семантическая разметка, `<picture>` для адаптивных изображений
- CSS3 — Flexbox, CSS-переменные, псевдоклассы, transitions
- Bootstrap Grid — 12-колоночная сетка (`container`, `row`, `col-*`)
- Адаптивность — mobile-first, брейкпоинты 320 / 576 / 768 / 1200 px
- Доступность — `aria-label`, фокус-состояния, `alt` у изображений
- Кроссбраузерность — Autoprefixer, Normalize.css

## Секции страницы

1. **Шапка** — логотип и навигация с якорными ссылками
2. **Баннер** — заголовок, описание, CTA-кнопка
3. **Преимущества** — три карточки с иконками
4. **Преподаватели** — сетка карточек с адаптивными фото (`<picture>`)
5. **Курсы** — четыре карточки курсов с hover-эффектами
6. **Подвал** — соцсети и копирайт
